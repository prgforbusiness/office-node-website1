# オフィス・ノード ウェブサイト

オフィス・ノードの公式ウェブサイトプロジェクトです。

## 事業概要

**事業者名**: オフィス・ノード  
**事業コンセプト**: 生成AI × 中小企業診断士  
**事業領域**: 中小企業の社員向け生成AI活用支援・伴走型DX支援

## プロジェクト概要

このプロジェクトは、中小企業の社員一人ひとりが生成AIを自己の業務に自由自在に活用できるようになることを支援するためのウェブサイトです。

## 技術スタック

- **フレームワーク**: Next.js（Reactベース、SSR対応）
- **スタイリング**: Tailwind CSS
- **UIライブラリ**: shadcn/ui
- **ホスティング**: Vercel（予定）

## プロジェクト構成

詳細なサイト構築プランは `WEBSITE_PLAN.md` を参照してください。

## 開発環境セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

## ライセンス

このプロジェクトはオフィス・ノードの所有物です。
